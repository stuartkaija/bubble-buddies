
# Bubble Buddies

A friend-finding application for scuba divers!


## Tech Stack and Dependencies

Bubble Buddies was built with React and Sass on the front end, and Node.js and Express on the backend. 

Other dependencies include:

React Router DOM - https://www.npmjs.com/package/react-router-dom

Axios - https://www.npmjs.com/package/axios

CORS - https://www.npmjs.com/package/cors

Uniqid - https://www.npmjs.com/package/uniqid

bcrypt - https://www.npmjs.com/package/bcrypt

JSON Web Token - https://www.npmjs.com/package/jsonwebtoken

dotenv - https://www.npmjs.com/package/dotenv

React Tinder Card - https://www.npmjs.com/package/react-tinder-card

## API

Bubble Buddies utilizes the World Weather Online API for weather data, and the Weather API for marine weather data. The links for these APIs can be found below.

https://www.worldweatheronline.com/developer/

https://www.weatherapi.com/api-explorer.aspx

## Try it for yourself!

Clone the repo to your local machine and install the dependencies to try Bubble Buddies for yourself. Weather and marine data will require API keys as described above.
